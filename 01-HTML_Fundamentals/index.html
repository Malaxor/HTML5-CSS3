The Basic Language of the web: HTML

All modern websites and web applications are built with three fundamental technologies: HTML, CSS and JavaScript, the languages of the web.

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for Hyper Text Markup Language. It's a markup language, not a programming language, used by web developers to structure and describe the content of a webpage.

HTML consists of elements that describe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.
